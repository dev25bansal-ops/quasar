# Spinning Cube Demo

A minimal 3D rendering demo for the Quasar Engine.

## Features

- Basic 3D cube rendering
- Rotation animation
- WASM/WebGPU support
- Editor toggle (F12)

## Running

```bash
cargo run -p spinning-cube
```

## Web

```bash
cd examples/spinning_cube
trunk build --release
```

## Controls

- **ESC** - Exit
- **F12** - Toggle editor
