# Physics Sandbox

A physics playground demonstrating rigid body simulation.

## Features

- Dynamic rigid bodies
- Static colliders
- Joint constraints
- Ray casting
- Debug visualization

## Running

```bash
cargo run -p physics-sandbox
```

## Controls

- **Click** - Spawn cube
- **Right-click** - Spawn sphere
- **WASD** - Move camera
- **Mouse** - Look around
- **F12** - Toggle editor

## Physics Debug

Press **F3** to toggle physics debug visualization:

- Collider wireframes
- Contact points
- Velocity vectors
