# Audio Demo

Demonstrates the Quasar Engine audio system.

## Features

- Sound playback
- 3D spatial audio
- DSP effects
- Music streaming
- Reverb zones

## Running

```bash
cargo run -p audio-demo
```

## Controls

- **1-5** - Play test sounds
- **Space** - Toggle music
- **F12** - Toggle editor

## Adding Audio Files

Place audio files in `assets/audio/`:

- `ambient.ogg` - Ambient sound
- `music.ogg` - Background music
- `sfx.wav` - Sound effects

## Supported Formats

- OGG Vorbis
- WAV
- MP3
- FLAC
