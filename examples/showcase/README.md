# Showcase Demo

A multi-feature demo showcasing Quasar Engine capabilities.

## Features

- Multiple 3D shapes
- Physics simulation
- Scene graph hierarchy
- Lighting and shadows
- Post-processing effects

## Running

```bash
cargo run -p showcase
```

## Controls

- **WASD** - Move camera
- **Mouse** - Look around
- **Space** - Jump
- **F12** - Toggle editor

## Requirements

- GPU with Vulkan/Metal/DX12 support
- 4GB RAM recommended
